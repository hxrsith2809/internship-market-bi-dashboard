# internship-market-bi-dashboard
An interactive Business Intelligence dashboard built in Jupyter using Python and Plotly to analyze internship hiring trends, skill demand, salary distribution, and remote work patterns.
This project presents an interactive Business Intelligence dashboard analyzing internship and entry-level job postings. The dashboard uncovers insights into hiring trends, in-demand skills, salary distributions, remote opportunities, and company-level hiring patterns using Python-based analytics.

The project demonstrates the complete BI workflow, including data generation, KPI calculation, aggregation, and interactive data visualization, all implemented using free and open-source tools
